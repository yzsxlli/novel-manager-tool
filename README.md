**小说文件批量管理工具**

**一个帮你整理乱七八糟小说文件夹的小工具**
https://github.com/yzsxlli/novel-manager-tool/blob/main/%E4%B8%BB%E9%A1%B5.png


**📖 简介**

**下载的小说文件夹乱七八糟，同一本书有好几个版本，分不清哪个是最新的？这个工具就是专门解决这个问题的。**



**自动识别文件名中的书名、作者、更新号，帮你把重复的文件清理掉，只保留最新的那个。**



**✨ 功能特性**

**🔍 智能识别**

**自动从文件名提取书名、作者、更新号**



**支持 .txt 和 .epub 格式**



**识别「完结」标签和「更XX」更新号**



**自动过滤非小说文件**



**🗑️ 批量清理**

**连载模式：按更新号排序（数字大的更新），更新号相同的按修改时间排序**



**完结模式：按修改时间排序，保留最新的文件**



**按「书名+作者」智能分组，只保留每组最新的文件**



**删除前可手动备份到 novel\_backups 文件夹**



**🛡️ 安全保障**

**测试模式：先预览会删除哪些文件，确认后再执行**



**手动备份：处理前可点击「备份文件」按钮备份全部文件**



**操作日志：详细记录每一步处理结果**



**删除确认：正式删除前二次确认**



**📝 便捷功能**

**历史记录：自动保存最近10次操作，双击即可复用**



**一键打开备份：点击「打开备份」快速查看备份文件**



**实时统计：显示总文件数、分组数、删除数、保留数**



**清空日志：随时清理日志显示区**



**🚀 快速开始**

**下载运行**

**从 Releases 下载最新版 小说批量管理工具.exe**



**双击运行**



**选择要整理的文件夹，开始使用**



**使用步骤**

**text**

**1. 点击「浏览」选择目标文件夹**

**2. 点击「备份文件」先备份（可选但推荐）**

**3. 选择处理模式（连载/完结）**

**4. 点击「测试识别」预览效果**

**5. 确认无误后点击「开始处理」**

**6. 处理完成后可点击「打开备份」查看备份文件**

**📝 处理逻辑说明**

**模式	排序规则	保留策略**

**连载	更新号 > 修改时间	保留更新号最大的文件，更新号相同则保留最新的**

**完结	修改时间	保留最新的文件**

**📝 使用场景举例**

**比如你的文件夹里有这些文件：**



**text**

**\[XXX 更107] SDFSD 作者：XX.txt**

**\[XXX 更107] SDFSD BY：XX.txt**

**\[XXX 完结] SDFSD BY：XX.txt**



**运行工具后：**



**连载模式会保留 更121 的版本**



**完结模式会保留带「完结」标签的版本**



**多余的文件会被清理掉，只留一个**
